类型

Typescript 的基本类型有：

  • string
  • number
  • boolean
  • symbol
  • null
  • undefined
  • any
  • unknown
  • never
  • void

string, number, boolean, symbol

string、number、boolean、symbol对应 Javascript 中包装类String、Number、Boolean、Symbol的区别。但是前者是类型，后者本质上是构造函数。

在编程中，应始终使用前者来表示变量的类型。

null, undefined

null和undefined被单独作为变量类型，与其它类型并不兼容。null表示空指针，但是不能将null理解为其它面向对象语言中未实例化的空变量。

any

当一个变量的类型是任意或无法确定时，我们可以声明它为any类型。

any类型变量可以赋值给任意类型的变量，也可以被赋值为任意类型变量。可以读取它的任意属性（即使不存在），读取的属性也是any类型。

编译器不会对any类型变量做类型检查，因此有时候可能有风险，在一些语法检查工具（如 ESLint）常常不允许使用any。
